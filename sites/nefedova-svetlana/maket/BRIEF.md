## Технические требования

- Чистый HTML + CSS + JS, без фреймворков
- Mobile-first CSS-архитектура (даже если сейчас только десктоп)
- Шрифты: Inter + Space Mono через Google Fonts
- Анимации: всегда с префиксами `-webkit-`
- SVG/изображения: явные `width` и `height`
- Safari: нет `inset: 0`, нет `color-mix()` без fallback
