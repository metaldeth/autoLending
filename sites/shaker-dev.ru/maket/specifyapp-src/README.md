# specifyapp.com — скачанный код для референса

Источник: https://specifyapp.com/

Скачано через `download_refs.py` (Playwright). Использовать только для изучения:
- структуры HTML
- анимаций (умеренный уровень)
- CSS (тёмная тема, 3D-визуал, схематичные линии)
- организации JS (Next.js, чанки)

**Не копировать код напрямую** — возможны лицензионные ограничения.
