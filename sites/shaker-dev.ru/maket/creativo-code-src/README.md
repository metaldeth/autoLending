# creativo-code.com — скачанный код для референса

Источник: https://www.creativo-code.com/

Скачано через `download_refs.py` (Playwright). Использовать только для изучения:
- структуры HTML
- анимаций и переходов
- CSS-приёмов (тёмная тема, типографика)
- организации JS (Next.js, webpack-чанки)

**Не копировать код напрямую** — возможны лицензионные ограничения.
