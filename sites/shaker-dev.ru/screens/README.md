# Скриншоты макета для согласования

Верхнеуровневые фреймы из `maket/shaker-dev.ru.pen` для отправки заказчику.

## Список фреймов

| № | Имя файла | Описание |
|---|-----------|----------|
| 1 | `01-dark-desktop.png` | **Dark Theme — Desktop** — десктоп, тёмная тема (1440×4512) |
| 2 | `02-light-desktop.png` | **Light Theme — Desktop** — десктоп, светлая тема (1440×4512) |
| 3 | `03-dark-mobile.png` | **Dark Theme — Mobile** — мобильная версия, тёмная тема (390×3460) |
| 4 | `04-light-mobile.png` | **Light Theme — Mobile** — мобильная версия, светлая тема (390×3460) |
| 5 | `05-theme-annotation.png` | **Theme Switcher — Аннотация** — описание переключателя тем и палитры |

## Как сохранить скриншоты сюда

Скриншоты были сгенерированы в чате Cursor. Сохраните каждый из пяти изображений из ответа ассистента в эту папку с именами выше (правый клик по картинке → «Сохранить изображение как…»).

После сохранения все файлы будут в `sites/shaker-dev.ru/screens/` — можно отправить заказчику.
